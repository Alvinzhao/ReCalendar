Concept

Maniana is a simple and fun app to manage your todo list. Maniana provides two side by side pages:
  • TODAY page for today's tasks.
  • MANIANA page for future tasks.
Every morning you will find all the tasks in the TODAY page. Send the tasks you don't plan to do today back to the MANIANA page by clicking on their arrows.

Page Switching:

Fling the screen right/left to switch between the TODAY and MANIANA pages.

Page Options

Add new task using a keyboard.
Add new task using voice recognition (enable/disable in Settings).
Reorder tasks in this age. Long press also deletes completed tasks (undo available).
Undo last delete operation in this page.
Send the task to the other page.
Indicates a locked task.
(Android 4.0 and up only) Menu button, in case your phone does not have a dedicated Menu button.

Task Menu Options

Click on a task to open its menu:

Toggle the task's done/todo status.
Edit the task's text.
Lock/Unlock the tasks. Locked tasks stay in the Maniana page until the lock expire.
Delete the task (undo available).

Task Coloring

Click on the very left end of a task to toggle its color.

Task Ordering

Long press on a task and drag it up/down.

Widgets

Maniana comes with two kinds of widgets that you can embed in your phone's home screen:
  • List Widgets - show TODAY's page active tasks. Come in four different sizes.
  • Icon Widget - shows the number of TODAY's page active tasks.
Clicking on a widget opens the TODAY page. You can control the appearance of the list widgets in the Maniana Settings screen (see below).

The toolbar at the top of the widget contains shortcut icons for adding new tasks using the keyboard or by voice. The toolbar can be disabled in Settings.

Settings

SOUND

  • Sound - toggles sound on/off.
  • Applause - controls the applause when a task is completed.

BEHAVIOR

  • Startup Animation - enables page scroll animation when the app starts.
  • Detailed Messages - enables more detailed response messages (for new users).
  • Auto Task Sort - when checked, tasks are always ordered with pending tasks before completed tasks before locked tasks.
  • Daily Task Cleanup - if checked, every night, all the completed tasks are deleted automatically.
  • Locks Expiration - sets locks automatic expiration (weekly, monthly or never).

PAGE APPEARANCE

  • Background Type - selects paper vs solid color background.
  • Background Color - selects background color (not available with paper background).
  • Font - selects task font.
  • Text Size - selects task text size.
  • Text Color - selects text color of non completed tasks.
  • Completed Text Color - selects text color of completed tasks.
  • Task Divider Color - selects color of lines between tasks (can be made transparent).

LIST WIDGET

  • Background Color - selects the list widgets background color. Can be made partially or fully transparent.
  • Text Color - selects list widgets text color.
  • Single Line - If checked, tasks are limited to a single line of text, otherwise, a second line is used if needed.
  • Show Toolbar - If checked, the toolbar with the shortcut icons is displayed at the top of the list widget.

MISCELLANEOUS

  • Release Info - click to see version and release information.
  • Tell A Friend - love Maniana? click to tell a friend about it.
  • Send Us Feedback - click to send Maniana developers feedback, requests, questions or suggestions.
  • Reset Settings - click to restores default settings (after confirmation). Does not affect task data.
