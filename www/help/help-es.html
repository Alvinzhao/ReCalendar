Concepto

Mañana es una aplicación para administrar listas de tareas, de manera sencilla y divertida. Mañana te provee de dos páginas, una al lado de la otra:
  • HOY Página para las tareas de hoy.
  • MAÑANA para las Tareas que postergo.
﻿Cada mañana encontrarás todas las tareas en la página HOY. Puedes enviar las tareas que no vas a hacer hoy a la página MAÑANA haciendo clic en las flechas.

Deslizar páginas:

Deslizar la pantalla a derecha o izquierda para alternar entre las páginas de HOY y MAÑANA.

﻿Opciones del Menú de Tareas

Seleccione una tarea para abrir el menú:

Toggle the task's done/todo status.
Edit the task's text.
Lock/Unlock the tasks. Locked tasks stay in the Maniana page until the lock expire.
Delete the task (undo available).

Colorear Tareas

Toque en el extremo izquierdo de una tarea para cambiar el color.

Ordenar Tareas

Long press on a task and drag it up/down.

Widgets

Mañana incluye dos tipos de widgets que se pueden incrustar en la pantalla de inicio de tu teléfono:
  • Widgets de Lista - muestra las tareas activas de HOY. Se pueden seleccionar cuatro tamaños diferentes.
  • Widget Icono - muestra el número de tareas activas para HOY.
Tocando un widget, se abre la página de HOY. Puede cambiar la apariencia del widget Lista en la pantalla de configuración de Mañana (ver más abajo).

La barra de herramientas en la parte superior del widget contiene iconos de acceso directo para agregar nuevas tareas mediante el teclado o por voz. Puede desactivar la barra de herramientas en las opciones de configuración.

Ajustes

SONIDO

  • Sonido - alterna entre sonido encendido y apagado.
  • Aplausos - controla los aplausos cuando una tarea se ha completado.

COMPORTAMIENTO

  • Animación de inicio - animación de desplazamiento de página cuando se inicia la aplicación.
  • Mensajes detallados - habilita mensajes de respuesta detallados (para usuarios nuevos).
  • Notificación diaria - al habilitarla, Mañana genera una notificación de Android diaria de las tareas activas (si hubiere).
  • Notificación de LED - si esta marcada, además de las notificaciones diarias, también se notifica con el dispositivo LED. (si está disponible).
  • Acción de agitación - establece una acción predeterminada para Mañana, al agitar el dispositivo.

TAREAS

  • Auto Task Sort - when checked, tasks are always ordered with pending tasks before completed tasks before locked tasks.
  • Add To Top - determines if new tasks are added to the top or bottom of the list.
  • Default Task Color - sets the initial color of new tasks.
  • Daily Task Cleanup - if checked, every night, all the completed tasks are deleted automatically.
  • Lock Expiration - sets locks automatic expiration (weekly, monthly or never).

PAGES

  • Set By Theme - click to select one of the predefined page appearances. You can then customize it using the settings below.
  • Paper - check to use paper background, uncheck for solid background color.
  • Paper/Background Color - selects paper or background color respectively (depends on Paper). Background color can be semi transparent if you like.
  • Icons - selects page icons.
  • Task Divider Color - selects color of lines between tasks (can be made transparent).
  • Title Text - selects font, size and color of page title text.
  • Task Text - select font, size and color of task text.

LIST WIDGET

  • Set By Theme - click to select one of the predefined widget appearances. You can then customize it using the settings below.
  • Paper - check to use paper background, uncheck for solid background color.
  • Paper/Background Color - selects paper or background color respectively (depends on Paper). Background color can be semi transparent if you like.
  • Show Toolbar - if checked, the toolbar with the shortcut icons is displayed at the top of the list widget.
  • Task Text - selects task text font, size and color.

BACKUP (EXPERIMENTAL)

  • Your Gmail Address - enter your Gmail address. Backup attachments will be email to this address.
  • Do Backup - click to perform task list backup. This entry is disabled until Gmail address is entered.
  • Do Restore - click for restore instructions.

MISCELLANEOUS

  • Release Info - click to see version and release information.
  • Tell A Friend - love Maniana? click to tell a friend about it.
  • Send Us Feedback - click to send Maniana developers feedback, requests, questions or suggestions.
  • Reset All Settings - click to restores default settings (after confirmation). Does not affect task data.
