Концепция

Maniana is a simple and fun app to manage your todo list. Maniana provides two side by side pages:
  • СЕГОДНЯ: страница для задач на сегодня.
  • MANIANA: страница для будущих заданий.
Every morning you will find all the tasks in the TODAY page. Send the tasks you don't plan to do today back to the MANIANA page by clicking on their arrows.

Переключение страниц:

Fling the screen right/left to switch between the TODAY and MANIANA pages.

Опции меню задач

Нажмите на задачу, чтобы открыть всплывающее меню:

Toggle the task's done/todo status.
Измените текст задачи.
Lock/Unlock the tasks. Locked tasks stay in the Maniana page until the lock expire.
Удалите задачу (возможна отмена действия).

Цвет задач

Нажмите на самый левый край задачи для переключения цвета.

Порядок задач

Long press on a task and drag it up/down.

Виджеты

Maniana comes with two kinds of widgets that you can embed in your phone's home screen:
  • List Widgets - show TODAY's page active tasks. Come in four different sizes.
  • Icon Widget - shows the number of TODAY's page active tasks.
Clicking on a widget opens the TODAY page. You can control the appearance of the list widgets in the Maniana Settings screen (see below).

The toolbar at the top of the widget contains shortcut icons for adding new tasks using the keyboard or by voice. The toolbar can be disabled in Settings.

Настройки

ЗВУК

  • Звук — включает или выключает звук.
  • Applause - controls the applause when a task is completed.

ПОВЕДЕНИЕ

  • Startup Animation - enables page scroll animation when the app starts.
  • Detailed Messages - enables more detailed response messages (for new users).
  • Daily Notification - if checked, Maniana issues a daily Android notification if active tasks are pending.
  • Notification LED - when true, daily notifications will also activate the device notification LED, if available.
  • Shaking Action - sets a Maniana page action to perform when the device is shaken.

ЗАДАЧИ

  • Автоматическая сортировка задач — если этот флажок установлен, задачи всегда добавляются в список до завершенных и заблокированных задач.
  • Добавлять сверху — определяет, будут новые задачи добавляются в верхнюю или нижнюю часть списка.
  • Цвет задач по умолчанию — устанавливает первоначальный цвет новых задач.
  • Ежедневная очистка задач — если флажок установлен, каждый вечер все завершённые задачи будут удаляются.
  • Срок действия блокировки — устанавливает срок автоматического истечения блокировки (через неделю, месяц или никогда).

СТРАНИЦЫ

  • Set By Theme - click to select one of the predefined page appearances. You can then customize it using the settings below.
  • Paper - check to use paper background, uncheck for solid background color.
  • Paper/Background Color - selects paper or background color respectively (depends on Paper). Background color can be semi transparent if you like.
  • Icons - selects page icons.
  • Task Divider Color - selects color of lines between tasks (can be made transparent).
  • Title Text - selects font, size and color of page title text.
  • Task Text - select font, size and color of task text.

LIST WIDGET

  • Set By Theme - click to select one of the predefined widget appearances. You can then customize it using the settings below.
  • Paper - check to use paper background, uncheck for solid background color.
  • Paper/Background Color - selects paper or background color respectively (depends on Paper). Background color can be semi transparent if you like.
  • Show Toolbar - if checked, the toolbar with the shortcut icons is displayed at the top of the list widget.
  • Task Text - selects task text font, size and color.

BACKUP (EXPERIMENTAL)

  • Your Gmail Address - enter your Gmail address. Backup attachments will be email to this address.
  • Do Backup - click to perform task list backup. This entry is disabled until Gmail address is entered.
  • Do Restore - click for restore instructions.

MISCELLANEOUS

  • Release Info - click to see version and release information.
  • Tell A Friend - love Maniana? click to tell a friend about it.
  • Send Us Feedback - click to send Maniana developers feedback, requests, questions or suggestions.
  • Reset All Settings - click to restores default settings (after confirmation). Does not affect task data.

USING THE COLOR PICKER

2. Выберите оттенок.
3. Выберите прозрачность (если нужно).
4. Нажмите, чтобы применить цвет.
